## üì¶ **Installation Instructions**

### **For Existing Installations**

```bash
# Pull latest changes
git pull origin main

# Install dependencies
composer install
npm install

# Run new migrations
php artisan migrate

# Clear caches
php artisan config:clear
php artisan cache:clear
php artisan view:clear

# Install dependencies

npm install

npm run build

php artisan serve
```
